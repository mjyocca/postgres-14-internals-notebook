## Docker Setup

```bash
docker compose up -d
```

## PGAdmin Setup

1. Login to pgAdmin at localhost:8888
2. Click "Add New Server" > Connection
- Hostname: `postgres`
- Port: `5432`
- Maintenance database: `postgres`
- Username: `postgres`
- Password: `postgres`
3. Create > Database
- Provide a name for `Database`
